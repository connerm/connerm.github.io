Website Architecture

Conner Mconkey published on 2020-06-04 included in Technology
This blog is a static website, meaning it is made only of simple HTML, CSS, and JavaScipt files. There is no back end server to process or store information. This limits the scope of what is possible here, but affords simplicity and security. It is hosted on Github Pages, as a user page. Github allows users to host a static website for their user profile. The site is automatically hosted when the user creates a repository named as their username, followed by github.
